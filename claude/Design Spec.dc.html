BEUNIQUE · BROWSER EXTENSION

Workspace UI — Design Specification

An implementation-ready spec for the extension's three working tabs — Recommendations, Layout Ideas, and Animations. Compact, action-oriented, and built for an in-browser design-analysis workspace. No marketing surfaces.

Full-tab workspace Left sidebar nav Light + dark Single-select direction · multi-select fixes
Principle
Show what was detected, what to fix, and which direction to choose — in that order.
Density
Dense but readable. No oversized type, no decorative blobs, gradients, or glass.
Output
Every selection assembles one copyable prompt. Never expose raw debug data.

04

Layout Ideas

Context line: "Choose a structure for this selected UI." A segmented control switches Layouts / Templates / Animations. Lives only in this tab — never duplicated in the sidebar. Switching segments never clears a selection.

Layout card
  • Abstract wireframe preview
  • Name + category chip
  • "Why it fits" line
  • "Best for" line
  • Selected = 2px ink ring + check
  • Action: Use layout → Selected
Template card
  • Representative wireframe
  • Name + category chip
  • Source / reference (e.g. "Inspired by Linear")
  • Reason it matches
  • Buttons: Use template · Open reference
Animations segment (page-level motion)

Three macro presets — Page entrance, Tab transition, Scroll reveal — set the layout's overall rhythm with a live preview. A banner points to the dedicated Animations tab for per-element motion. Selectable like a layout (single).

06

Interaction model & states

Selection rules
  • Fixes — multi-select via checkbox
  • Layout, Template, Animation — one each; tap again to clear
  • Selections persist across tab and segment changes
  • Selected state is always explicit: filled box or 2px ink ring + check
Prompt assembly
  • Bottom bar mirrors current selection at all times
  • "Preview prompt" reveals the assembled text drawer
  • "Copy" writes it to clipboard → "Copied ✓" for 1.5s
  • Prompt = selected fixes + chosen layout / template / motion
Empty & system states

  • Error — inline retry, never a raw stack trace
